## 📥 How to Download & Install

### Method 1: Copy Command from Website

1. Browse the marketplace at https://neuron-models.vercel.app
2. Click **"📥 Download & Install"** on any model
3. A modal appears with installation instructions
4. Click **"Copy"** to copy the command
5. Open PowerShell as Administrator
6. Paste the command and press Enter
7. Model installs automatically!

### Method 2: From Command Line

```powershell
neuron download-model
```

This opens the marketplace website in your default browser.

### Method 3: Direct Installation

If you know the model name:

```powershell
neuron download-model neuron-creative
```

This downloads and installs the model directly without opening the browser.
